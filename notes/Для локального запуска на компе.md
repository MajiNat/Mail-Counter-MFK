В папке, где находиться сама прога, в пути написать "cmd", далее ввести подряд команды:

***npm install***

***npm run dev***

Приложение откроется в браузере по адресу, пример http://localhost:0000

Чтобы перейти, нужно зажать CTRL+ЛКМ



Для корректного запуска сайта нужно скачать последнюю версию Node.js https://nodejs.org/en/download
